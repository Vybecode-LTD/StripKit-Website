Everything in the box

Built for plugin asset production

From a single source image to a shippable, manifest-bound sprite sheet — without leaving the app.

⟳

Rotary knobs

Sweep 270°, 300°, 360° or any custom range, rotated about the knob's true centre so it spins clean — no wobble.

↕

Faders & sliders

Vertical faders and horizontal sliders: the cap travels its track with pixel-accurate end margins.

▮▮▮

Meters

Procedural LED segments or layered on/off art, four fill directions, discrete or continuous.

◎

Smart alignment

Auto-detects the spin centre on load; a draggable crosshair lets you nail it by eye on a smooth 1024-step preview.

⇊

Filmstrip importer

Drop an existing strip — StripKit detects its frame count and orientation, extracts a frame, or re-stacks it.

▦

Batch processing

Point it at a folder and render a whole set of sources to strips in one off-thread pass, with progress and cancel.

{ }

skin.json manifest

Emit a schema-valid manifest that binds each strip to a parameter id — ready for a JUCE LookAndFeel loader.

✦

Crisp rendering

Supersampling + a Mitchell cubic resampler keep rotated edges sharp; export @2x for HiDPI in one toggle.

▷

Live preview

Scrub, play, or step frame-by-frame and watch the control animate before you ever hit export.
